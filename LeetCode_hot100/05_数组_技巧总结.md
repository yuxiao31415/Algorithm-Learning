## List<Integer> 转 int[] 
```java
List<Integer> list = Arrays.asList(1, 2, 3, 4);
int[] arr = list.stream().mapToInt(Integer::intValue).toArray();
// Arrays.asList(T ...)
//list.toArray(T )方法只能转成 Integer[]
//list.stream().toArray() 转成Object[]
//list.stream().toArray(Integer[]::new) 转成Integer[]
```
## int[] 转 List<Integer> 
```java
int[] arr = {1, 2, 3, 4};
List<Integer> list = Arrays.stream(arr).boxed().toList();   // Java 16+
// Arrays.stream(arr) 生成 IntStream
// list.stream() 生成 Stream<Integer>
```


## List<int[]> 转 int[][] 
```java
int[][] arr = list.toArray(new int[][]); //toArray() 
```
## int[][] 转 List<int[]> 
```java
List<int[]> list = Arrays.asList(arr); 
```
## Arrays.sort()排序
```java
Arrays.sort(matrix,(a,b) -> a[0]-b[0]);// a[0]-b[0]<0 a在前
                                       // b[0]-a[0]<0 a在前 但是此时a[0]>b[0] 所以大的在前
如果返回负数 → a排前面
Arrays.sort(matrix,(a,b) -> Integer.compare(a[0],b[0]));
```

## stream流   

一、Stream 创建方式  

| 创建方式 | 示例代码 | 说明 |
|---|---|---|
| Collection 创建 | `list.stream()` | 最常见，从集合创建流 |
| 并行流 | `list.parallelStream()` | 多线程并行处理 |
| 数组创建 | `Arrays.stream(arr)` | 从数组创建 |
| Stream.of | `Stream.of(1,2,3)` | 直接创建流 |
| 文件流 | `Files.lines(path)` | 按行读取文件 |
| 生成无限流 | `Stream.generate(() -> 1)` | 无限生成 |
| 迭代生成 | `Stream.iterate(0,n->n+1)` | 按规则生成 |  

二、中间操作  
| 方法 | 作用 | 示例 |
|---|---|---|
| `filter()` | 过滤元素 | `stream.filter(x -> x > 10)` |
| `map()` | 元素转换 | `stream.map(x -> x * 2)` |
| `flatMap()` | 展开流 | `stream.flatMap(List::stream)` |
| `distinct()` | 去重 | `stream.distinct()` |
| `sorted()` | 排序 | `stream.sorted()` |
| `sorted(Comparator)` | 自定义排序 | `stream.sorted((a,b)->b-a)` |
| `limit(n)` | 取前 n 个 | `stream.limit(5)` |
| `skip(n)` | 跳过前 n 个 | `stream.skip(2)` |
| `peek()` | 调试查看 | `stream.peek(System.out::println)` |  

三、终止操作   

| 方法 | 作用 | 示例 |
|---|---|---|
| `forEach()` | 遍历 | `stream.forEach(System.out::println)` |
| `collect()` | 收集成集合 | `stream.collect(Collectors.toList())` |
| `toArray()` | 转数组 | `stream.toArray()` |
| `reduce()` | 归约计算 | `stream.reduce(Integer::sum)` |
| `count()` | 统计数量 | `stream.count()` |
| `findFirst()` | 找第一个 | `stream.findFirst()` |
| `findAny()` | 任意一个 | `stream.findAny()` |
| `anyMatch()` | 是否存在匹配 | `stream.anyMatch(x -> x > 10)` |
| `allMatch()` | 是否全部匹配 | `stream.allMatch(x -> x > 10)` |
| `noneMatch()` | 是否都不匹配 | `stream.noneMatch(x -> x > 10)` |
| `min()` | 最小值 | `stream.min(Integer::compare)` |
| `max()` | 最大值 | `stream.max(Integer::compare)` |  

四、Collectors 常见用法

`collect()` 通常配合 `Collectors` 使用。

| 方法 | 作用 | 示例 |
|---|---|---|
| `toList()` | 转 List | `collect(Collectors.toList())` |
| `toSet()` | 转 Set | `collect(Collectors.toSet())` |
| `toMap()` | 转 Map | `collect(Collectors.toMap(k,v))` |
| `joining()` | 拼接字符串 | `collect(Collectors.joining(","))` |
| `groupingBy()` | 分组 | `collect(Collectors.groupingBy(User::getAge))` |
| `partitioningBy()` | 条件分区 | `collect(Collectors.partitioningBy(x -> x > 60))` |
| `counting()` | 计数 | `collect(Collectors.counting())` |
| `averagingInt()` | 求平均 | `collect(Collectors.averagingInt(User::getAge))` |
