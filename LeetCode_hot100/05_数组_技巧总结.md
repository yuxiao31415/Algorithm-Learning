## List<Integer> 转 int[] 
```java
List<Integer> list = Arrays.asList(1, 2, 3, 4);
int[] arr = list.stream().mapToInt(Integer::intValue).toArray();//toArray(T )只能转成 Integer[]
```
## int[] 转 List<Integer> 
```java
int[] arr = {1, 2, 3, 4};
List<Integer> list = Arrays.stream(arr).boxed().toList();   // Java 16+
```


## List<int[]> 转 int[][] 
```java
int[][] arr = list.toArray(new int[list.size()][]); //toArray() 
```
## int[][] 转 List<int[]> 
```java
List<int[]> list = Arrays.asList(arr); 
```
