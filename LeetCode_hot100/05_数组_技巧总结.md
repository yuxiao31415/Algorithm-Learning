## List<Integer> 转 int[] 
```java
List<Integer> list = Arrays.asList(1, 2, 3, 4);
int[] arr = list.stream().mapToInt(Integer::intValue).toArray();
// Arrays.asList(T ...)
//list.toArray(T )方法只能转成 Integer[]
//list.stream().toArray() 转成Object[]
//list.stream().toArray(Integer[]::new) 转成Integer[]
```
## int[] 转 List<Integer> 
```java
int[] arr = {1, 2, 3, 4};
List<Integer> list = Arrays.stream(arr).boxed().toList();   // Java 16+
// Arrays.stream(arr) 生成 IntStream
// list.stream() 生成 Stream<Integer>
```


## List<int[]> 转 int[][] 
```java
int[][] arr = list.toArray(new int[][]); //toArray() 
```
## int[][] 转 List<int[]> 
```java
List<int[]> list = Arrays.asList(arr); 
```
## Arrays.sort()排序
```java
Arrays.sort(matrix,(a,b) -> a[0]-b[0]);// a[0]-b[0]<0 a在前
                                       // b[0]-a[0]<0 a在前 但是此时a[0]>b[0] 所以大的在前
如果返回负数 → a排前面
Arrays.sort(matrix,(a,b) -> Integer.compare(a[0],b[0]));
```
# 方法引用
```java
1. Integer::sum         list.stream().reduce(0, Integer::sum);                          ->  list.stream().reduce(0, (a, b) -> a + b);
2. System.out::println  list.forEach(System.out::println);                              ->  list.forEach(x -> System.out.println(x));
3. String::length       list.stream().map(String::length).forEach(System.out::println); ->  .map(s -> s.length())
4. String::toUpperCase  list.stream().map(String::toUpperCase).toList();                ->  .map(s -> s.toUpperCase())
5. Integer::parseInt    list.stream().map(Integer::parseInt).toList();                  ->  .map(s -> Integer.parseInt(s))
6. String::valueOf      list.stream().map(String::valueOf).toList();                    ->  .map(x -> String.valueOf(x))
7. Comparator           list.sort(Comparator.comparingInt(Person::getAge));             ->  list.sort((a,b) -> a.getAge() - b.getAge());  
    .comparingInt(Person::getAge)
8. Collections::sort    listOfLists.forEach(Collections::sort);                         ->  listOfLists.forEach(l -> Collections.sort(l));
9. ArrayList::new       list.stream()                                                   ->  .collect(Collectors.toCollection(() -> new ArrayList<>()))
                          .collect(Collectors.toCollection(ArrayList::new));
10. Map.Entry::getKey   map.entrySet()                                                  -> .map(e -> e.getKey())
                         .stream()
                         .map(Map.Entry::getKey)
                         .toList();

```
