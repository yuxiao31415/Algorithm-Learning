## 81.爬楼梯

```java
class Solution {
    public int climbStairs(int n) {
        int[] f = new int[n+1];
        f[0]=f[1]=1;
        for(int i =2;i<=n;i++){
            f[i] = f[i-1]+f[i-2];
        }
        return f[n];

    }
}
```

## 82.杨辉三角**  
```java
class Solution {
    public List<List<Integer>> generate(int numRows) {
        List<List<Integer>> ans = new ArrayList<>();
        int[] row1 = new int[]{1};
        int[] row2 = new int[]{1,1};
        ans.add(Arrays.stream(row1).boxed().toList());
        if(numRows==1) return ans;
        ans.add(Arrays.stream(row2).boxed().toList());
        for(int i=3;i<=numRows;i++){
            int[] row = new int[i];
            row[0] = row[i-1]=1;
            for(int j=1;j<i-1;j++){
                row[j] = ans.getLast().get(j-1)+ans.getLast().get(j);
            }
            ans.add(Arrays.stream(row).boxed().toList());
        }
        return ans;

    }
}
```

## 83.打家劫舍  //////////////
```java
class Solution {
    public int rob(int[] nums) {
        int[] f = new int[nums.length+1];// f[i] 偷完i家之后最多有多少钱 f[n]
        f[0] = 0;
        f[1] = nums[0];
        for(int i=1;i<nums.length;i++){
            f[i+1] = Math.max(f[i],f[i-1]+nums[i]);
        }
        return f[nums.length];
    }
}
```
### 思路

```
第i位置的最优解,max(当前位置值+上上个位置的值,上个位置的值)
```
