class Solution {
    public int singleNumber(int[] nums) {
        int ans = 0;
        for(int i:nums){
            ans^=i;
        }

## 97.多数元素  
```java
class Solution {
    public int majorityElement(int[] nums) {
        int ans =0;
        int count = 0;
        for(int num:nums){
            if(count==0){
                ans = num;
                count =1;
            }else{
                count += num==ans?1:-1;
            }
        }
        return ans;
    }
}
```
### 思路
1. __擂主登场：__``nums[0]`` 成为初始擂主，生命值为 1。  
2. __挑战者出现：__遍历后续元素，作为挑战者。  
3. __比武：__如果挑战者与擂主属于同一门派（值相同），那么擂主生命值加 1，否则擂主生命值减 1。  
4. __擂主更迭：__如果比武后，擂主生命值降为 0（同归于尽），那么下一个挑战者成为新的擂主，生命值为 1。  
5. 最后在擂台上的那人，便是武林盟主（严格众数）。

## 99.下一个排序  
```java
//5 2 4 3 1找到第一个非单调增的位置,交换元素 5 3 4 2 1 -> 因为p1后面都是单调增,所以翻转变为单调减变化更小 6 2 1 4
class Solution {
    public void nextPermutation(int[] nums) {
        int n = nums.length;
        // 规则一 找到不单调的位置
        int p1 =n-2;
        while(p1>=0&&nums[p1]>=nums[p1+1]){
            p1--;
        }
        // 规则二 从后往前找,使得变化最小
        if(p1>=0){//说明找到了 若没找到则p1=-1;
            int p2 = n-1;
            while(nums[p2]<=nums[p1]){
                p2--;
            }
            // 找到后交换这两个数字
            swap(nums,p1,p2);
        }
        // 翻转从p1+1到最后
        reverse(nums,p1+1,n-1);
    }
    private void swap(int[] nums,int p1,int p2){
        int temp = nums[p1];
        nums[p1] = nums[p2];
        nums[p2] = temp;
    }
    private void reverse(int[] nums,int p1,int p2){
        while(p1<p2){
            swap(nums,p1++,p2--);
        }
    }

}
```
### 思路

1. __规则一:__先从右往左找,找到第一个 满足__右边存在大于本身的数__的数  
2. __规则二:__然后再从这个数的左往右找,找到__大于该数的结集合中最小的数__或者从右往左找,找到__第一个大于该数的__数  
3. 交换这两条规则找到的数  
4. 然后将该数位置之后的数进行翻转

## 100.寻找重复数  
```java
class Solution {
    public int findDuplicate(int[] nums) {
        int slow = 0; // 0 一定不在环上，适合作为起点
        int fast = 0;
        while (true) {
            slow = nums[slow]; // 等价于 slow = slow.next
            fast = nums[nums[fast]]; // 等价于 fast = fast.next.next
            if (fast == slow) { // 快慢指针移动到同一个节点
                break;
            }
        }
        int head =0;
        while(slow!=head){
            slow = nums[slow];
            head = nums[head];
        }
        return slow; // 入环口即重复元素
    }
}
```
### 思路

__注意__:题目说了整数范围为[1,n+1] ,减去一正好等于数组坐标范围  

所以可以当做环形链表来做  

1. 定义快慢指针找到相遇点  
2. 另慢指针从头开始  
3. 快指针步长变为一,找到相遇点
