## 96.只出现一次的元素
``` java
class Solution {
    public int singleNumber(int[] nums) {
        int ans = 0;
        for(int i:nums){
            ans^=i;
        }
        return ans;
    }
}
```
### 思路
任何数与本身的异或为0  
任何数与0的异或为该数本身
