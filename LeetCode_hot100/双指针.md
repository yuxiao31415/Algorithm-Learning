# hot100 - - 双指针
## 4. 移动零
``` java
class Solution {
    public void moveZeroes(int[] nums) {
        int fast = 1;
        int slow = 0;
        while(fast<nums.length){
            if(nums[slow]!=0){
                slow++;
            }
            if(nums[fast-1]==0&&nums[fast]!=0){
                //swap(nums[slow],nums[fast])
                nums[slow]=nums[fast];
                nums[fast] =0;
            }
            fast++;
        }
    }
}
```
### 思路
* 慢指针:零的位置  
* 快指针:前一个元素为零,本身位置不为零  
* 不断交换值
