## 基础题目

### 1. 斐波那契数
题目描述：计算第 n 个斐波那契数。

LeetCode链接：  
https://leetcode.cn/problems/fibonacci-number/

```java
class Solution {
  public static void mian(String[] args){
    Scanner sc = new Scanner();
    int n = sc.nextInt();
    if(n==0) return 0;
    int[] f= new int[n+1];
    f[0]=0;
    f[1]=1;
    for(int i=2;i<=n;i++){
         f[i] = f[i-1]+f[i-2];
    }
    System.out.println(f[n]);
}
}
```

### 2. 爬楼梯
题目描述：每次可以爬 1 或 2 个台阶，求爬到第 n 阶共有多少种方法。

LeetCode链接：  
https://leetcode.cn/problems/climbing-stairs/
```java
class Solution {
    public int climbStairs(int n) {
        if(n==0) return 0;
        int[] f = new int[n+1];
        f[0] = 1;
        f[1] = 1;
        for(int i=2;i<=n;i++){
            f[i] = f[i-1]+f[i-2];
        }
        return f[n];
    }
}
```

### 3. 使用最小花费爬楼梯
题目描述：每个台阶都有花费 cost，可以从第 0 或 1 阶开始，求到达楼顶的最小花费。

LeetCode链接：  
https://leetcode.cn/problems/min-cost-climbing-stairs/
```java
class Solution {
    public int minCostClimbingStairs(int[] cost) {
        int[] f = new int[cost.length+1];
        for(int i=2;i<=cost.length;i++){
            f[i] = Math.min(f[i-1]+cost[i-1],f[i-2]+cost[i-2]);
        }
        return f[cost.length];
    }
}
```

### 4. 不同路径
题目描述：机器人从左上角走到右下角，每次只能向右或向下，求不同路径数量。

LeetCode链接：  
https://leetcode.cn/problems/unique-paths/
```java
class Solution {
    public int uniquePaths(int m, int n) {
        int[][] f = new int[m+1][n+1];
        f[0][1] =1;
        for(int i=1;i<=m;i++){
            for(int j=1;j<=n;j++){
                f[i][j] = f[i-1][j]+f[i][j-1];
            }
        }
        return f[m][n];
    }
}
```

### 5. 不同路径 II
题目描述：网格中存在障碍物，求从左上角到右下角的不同路径数量。

LeetCode链接：  
https://leetcode.cn/problems/unique-paths-ii/
```java
class Solution {
    public int uniquePathsWithObstacles(int[][] obstacleGrid) {
        int m = obstacleGrid.length;
        int n = obstacleGrid[0].length;
        int f[][] =new int[m+1][n+1];
        f[0][1] =1;
        for(int i=1;i<=m;i++){
            for(int j=1;j<=n;j++){
                if(obstacleGrid[i-1][j-1]!=1){
                    f[i][j] =f[i-1][j]+f[i][j-1];
                }
            }
        }
        return f[m][n];
    }
}
```

### 6. 整数拆分
题目描述：将整数 n 拆分成至少两个正整数，使其乘积最大。

LeetCode链接：  
https://leetcode.cn/problems/integer-break/
```java
class Solution {
    public int integerBreak(int n) {
        int[] f = new int[n+1];
        f[0]=1;
        f[1] =1;
        for(int i=2;i<=n;i++){
            int max = 0;
            for(int j=1;j<i;j++){
                int x = Math.max(f[i-j],i-j);
                int y = Math.max(f[j],j);
                max = Math.max(max,x*y);
            }
            f[i] = max;
        }
        return f[n];
    }
}
```

### 7. 不同的二叉搜索树
题目描述：给定 n 个节点，求能组成多少种不同的二叉搜索树。

LeetCode链接：  
https://leetcode.cn/problems/unique-binary-search-trees/
```java
class Solution {
    public int numTrees(int n) {
        int[] f = new int[n+1];
        f[0] =1;
        f[1] =1;
        for(int i=2;i<=n;i++){
            int count =0;
            for(int j=0;j<i;j++){
                count+=f[j]*f[i-j-1];
            }
            f[i] = count;
        }
        return f[n];
    }
}
```
