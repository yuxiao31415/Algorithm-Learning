## 动态规划 - 背包问题

### 1. 分割等和子集

**题目描述：**  
给定一个只包含正整数的非空数组，判断是否可以将这个数组分割成两个子集，使得两个子集的元素和相等。

**LeetCode链接：**  
https://leetcode.cn/problems/partition-equal-subset-sum/  
**一维**
```java
class Solution {
    public boolean canPartition(int[] nums) {
        int sum = 0;
        for(int num : nums){
            sum += num;
        }
        if(sum % 2 != 0) return false;
        int target = sum / 2;
        boolean[] f = new boolean[target + 1];
        f[0] = true;
        for(int num : nums){ // 遍历物品
            for(int j = target; j >= num; j--){ 
            // 倒序遍历背包（01背包关键）
                f[j] = f[j] || f[j - num];
                // 不选num 或 选num
            }
        }
        return f[target];
    }
}
```
**二维**
```java
class Solution {
    public boolean canPartition(int[] nums) {
        int total = 0;
        int n = nums.length;
        for(int i=0;i<n;i++){
            total+=nums[i];
        }
        if(total%2!=0) return false;
        total = total/2;
        boolean[][] f = new boolean[n+1][total+1];// i个元素能否达到j值
        f[0][0]=true;
        for(int i = 1;i<=n;i++){
            for(int j=1;j<=total;j++){
                f[i][j] = f[i-1][j]||(j>=nums[i-1]&&f[i-1][j-nums[i-1]]);
            }
        }
        return f[n][total];

    }
}
```

### 2. 最后一块石头的重量 II

**题目描述：**  
每一回合从石头中选出两块最重的石头相撞，求最后剩下石头的最小可能重量。

**LeetCode链接：**  
https://leetcode.cn/problems/last-stone-weight-ii/    
**一维**
```java
class Solution {
    public int lastStoneWeightII(int[] stones) {
        int sum = 0;
        for(int s : stones){
            sum += s;
        }
        int target = sum / 2;
        int[] f = new int[target + 1]; // f[j] 表示容量 j 能得到的最大重量
        for(int stone : stones){ // 遍历物品
            for(int j = target; j >= stone; j--){ 
            // 倒序遍历（01背包关键）
                f[j] = Math.max(f[j], f[j - stone] + stone);
                // 不选 stone
                // 或者选 stone
            }
        }
        return sum - 2 * f[target];

    }
}
```
**二维**
```java
class Solution {
    public int lastStoneWeightII(int[] stones) {
        int sum = 0;
        for(int i:stones){
            sum+=i;
        }
        int total=sum/2;
        int[][] f = new int[stones.length+1][total+1];// i 个石头计算出的最接近j的 值
        for(int i=1;i<=stones.length;i++){
            for(int j=1;j<=total;j++){
                if(j>=stones[i-1]){
                    f[i][j] = Math.max(stones[i-1]+f[i-1][j-stones[i-1]],f[i-1][j]);
                }else{
                    f[i][j] = f[i-1][j];
                }
                
            }
        }
        return sum-2*f[stones.length][total];
    }
}
```

### 3. 目标和

**题目描述：**  
给定一个整数数组 nums 和一个整数 target，向数组中的每个整数前添加 '+' 或 '-'，求使表达式结果等于 target 的不同方法数。

**LeetCode链接：**  
https://leetcode.cn/problems/target-sum/  
```java
class Solution {
    public int findTargetSumWays(int[] nums, int target) {
        int sum = 0;
        for(int num : nums){
            sum += num;
        }
        if(Math.abs(target) > sum) return 0;
        if((sum + target) % 2 != 0) return 0;
        int size = (sum + target) / 2;
        int[] f = new int[size + 1];
        f[0] = 1;
        // 凑出0的方案数为1
        for(int num : nums){ // 遍历物品
            for(int j = size; j >= num; j--){ 
                f[j] += f[j - num];
                // 不选 num
                // 选 num
            }
        }
        return f[size];
    }
}
```
```java
class Solution {
    public int findTargetSumWays(int[] nums, int target) {
        // 关键点: 所有正数的和 与 所有负数的和 之和 为 target
        //      而 所有正数的和 与 所有负数的和 之差 为 sum
        //  所以即为 有多少中和为 (target+sum)/2
        int n = nums.length;
        int sum = 0;
        for(int i:nums) sum+=i;
        int size = (target+sum)/2;
        if((sum + target) % 2 != 0) return 0;
        if(Math.abs(target)>sum) return 0;
        int[][] f= new int[n+1][size+1];
        
        f[0][0] = 1;
        for(int i=1;i<=n;i++){
            for(int j=0;j<=size;j++){
                if(j>=nums[i-1]){
                    f[i][j] = f[i-1][j]+f[i-1][j-nums[i-1]];
                }else{
                     f[i][j] = f[i-1][j];
                }
                
            }
        }
        return f[n][size];
    }
}
```
