## 一维数据解决动态规划

### 01背包问题 VS. 完全背包

**01背包模版**  

**参考题目:** 目标和, https://leetcode.cn/problems/target-sum/  
```java
  int[] f = new int[target + 1];
  //外层物品,内层背包-->不考虑顺序 
   for(int i=0;i<nums.length;i++){ // 遍历物品
      for(int j = target; j >= nums[i]; j--){ // 倒序遍历（01背包关键）
           f[j] += f[j - nums[i]];
            // 不选 nums[i]
            // 或者选  nums[i]
            //  f[j] =f[j](不选) + f[j - nums[i]](选); 
      }
  }
  return f[target];
```
**完全背包模板**   

```java
// 第一种情况,最简单的完全背包
        int[] f = new int[amount + 1];
        //初始化为1
        f[0] = 1;
        // 先物品,后背包-->无排序
        for (int i = 0; i < coins.length; i++) {
            for (int j = coins[i]; j <= amount; j++) {
                f[j] += f[j - coins[i]];
            }
        }
        return dp[amount];
```

```java
// 第二种情况,完全背包+排序
        int[] f = new int[target+1];
        // 初始化为1
        f[0] =1;
        // 先背包,后物品 -->有排序 
        for(int i=1;i<=target;i++){
            for(int j=0;j<nums.length;j++){
                if(i>=nums[j]){
                    f[i] +=f[i-nums[j]];
                }
            }
        }
        return f[target];
```
```java
// 第三种情况,完全背包+最优化
        int[] f = new int[amount+1];
        // 初始化为最大值
        Arrays.fill(f,amount + 1);
        f[0] =0;
        // 先物品,后背包-->无排序
        for(int i=0;i<coins.length;i++){
            for(int j=1;j<=amount;j++){
                if(j>=coins[i]){
                    // 取最小值,且+1
                    f[j] = Math.min(f[j],f[j-coins[i]]+1);
                }
            }
        }
        // 返回时也要判断
        return f[amount]>amount?-1:f[amount];
```
