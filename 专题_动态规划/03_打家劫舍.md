## 动态规划 - 打家劫舍

### 1. 打家劫舍

**题目描述：**  
你是一个专业的小偷，计划偷窃沿街的房屋。每间房屋内都藏有一定金额的现金，相邻的房屋装有相互连通的防盗系统，如果两间相邻的房屋在同一晚上被小偷闯入，系统会自动报警。  
给定一个代表每个房屋存放金额的非负整数数组，计算你在不触动警报装置的情况下，一夜之内能够偷窃到的最高金额。

**LeetCode链接：**  
https://leetcode.cn/problems/house-robber/
```java
class Solution {
    public int rob(int[] nums) {
        int n = nums.length;
        int[] f = new int[n+1];
        f[1] = nums[0];
        for(int i=2;i<=n;i++){
            f[i] = Math.max(f[i-1],f[i-2]+nums[i-1]);
        }
        return f[n];
    }
}
```

### 2. 打家劫舍 II

**题目描述：**  
你是一个专业的小偷，计划偷窃沿街的房屋，每间房屋内都藏有一定金额。这个地方所有的房屋都围成一圈，这意味着第一个房屋和最后一个房屋是紧挨着的。  
相邻的房屋装有相互连通的防盗系统，如果两间相邻的房屋在同一晚上被小偷闯入，系统会自动报警。  
给定一个代表每个房屋存放金额的非负整数数组，计算你在不触动警报装置的情况下，一夜之内能够偷窃到的最高金额。

**LeetCode链接：**  
https://leetcode.cn/problems/house-robber-ii/
```java
class Solution {
    public int rob(int[] nums) {
        if(nums.length==1) return nums[0];
        return Math.max(rob(nums,0,nums.length-2),rob(nums,1,nums.length-1));
    }
    private int rob(int[] nums,int left,int right){
        int n = right-left+1;
        int[] f = new int[n+1];
        f[1] =nums[left];
        for(int i = 2;i<=n;i++){
            f[i] = Math.max(f[i-1],nums[i+left-1]+f[i-2]);
        }
        return f[n];
    }
}
```

### 3. 打家劫舍 III

**题目描述：**  
小偷又发现了一个新的可行窃的地区，这次所有的房屋排列成一棵二叉树。  
如果两个直接相连的房屋在同一天晚上被打劫，房屋将自动报警。  
给定二叉树的根节点 root ，返回在不触动警报的情况下，小偷能够盗取的最高金额。

**LeetCode链接：**  
https://leetcode.cn/problems/house-robber-iii/
```java
class Solution {
    public int rob(TreeNode root) {
        int[] res = robAction(root);
        return Math.max(res[0], res[1]);
    }

    int[] robAction(TreeNode root) {
        int res[] = new int[2];
        if (root == null)
            return res;

        int[] left = robAction(root.left);
        int[] right = robAction(root.right);

        res[0] = Math.max(left[0], left[1]) + Math.max(right[0], right[1]);
        res[1] = root.val + left[0] + right[0];
        return res;
    }

}
```
